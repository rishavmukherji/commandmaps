Use of Spatial Memory Eases Navigation

No more searching through different tabs and menus to find what you are looking for. No more hunting in different areas because the buttons automatically change places. CommandMaps is static and shows you the most important options all in one place - and they remain in the same place however many times to use it.

Hierarchy Flattening To Improve GUI Performance

Think of all the different tabs in Microsoft Word. CommandMaps flattens them all in one simple structure so that you can find what you need - when you need it.
